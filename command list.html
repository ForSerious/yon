Command What: Usage:
ifconfig Can tell you what your IP address is. Type it in like normal.
mkdir Makes a new folder. mkdir [name] – where name is the name of the folder you want to creat.
uniq Makes the output not repeat. Add it to the commands.
wc Displays how many words are in the output. Use it like normal.
xclip Puts what has been cut/coppied into the commandline. No special usage...
echo Prints what fallows to the command line as text. Good for seeing output.
cd Change Directory. cd [name] changes to the directory name.
tree Shows a tree representation of where files are. Type it in like normal.
file Displays information about a file. file [name] – where name is the file you want to know about.
type Like file, displays information about commands. type [command]
apropos Searches for key words in the manual. apropos [regular expression or keywords]
man Pops up the manual reader about the item in question. man [commmand]
pwd Stands for print working directory. Put it in like noraml.
cat Short for concatenate. Adds a file to the end of another and prints it to the screen. [file] cat [otherFile]
less A commandline based text viewer with lots of functionality. less [file] – Opens and reads the first few lines of file.
